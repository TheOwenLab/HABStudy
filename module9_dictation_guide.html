🎙️ Optional: Talk Instead of Type ▼
If you prefer not to type out long answers, you can use your computer's built-in voice-to-text tool to talk directly into the text boxes below. This is entirely optional and free.
🪟 Windows
  1. Click inside any text answer box below.
  2. Hold the Windows Logo ⊞ Key, then tap the H key.
  3. When the microphone widget appears, start speaking naturally.
🍏 Apple Mac
  1. Click inside any text answer box below.
  2. Press the Microphone 🎤 Key (F5) on your top row.
  3. Or, double-press the Globe 🌐 / Fn Key in the bottom-left corner.
  4. Speak naturally to fill out the box.
💡 Remember to say your punctuation marks out loud (for example: say "period" or "comma" to format sentences).
